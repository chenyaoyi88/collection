##26.1 获取常用地址列表
获取登陆用户的地址列表

默认加入当前登陆的用户id进行查询
###接口调用请求说明

说明
协议 http
http请求方式 GET
请求url http://[host]/api/v1/addressBookRest/list?start=0&limit=2&serviceType＝1
数据格式 无

###请求参数说明

参数 是否必须 说明
start 否 数据开始下标，从0开始（默认为0）
limit 否 查询条数，默认为50条
serviceType 是 广货宝地址服务类型:
1:物流地址
2:小件地址

###返回数据说明

参数 说明 类型
id 地址ID Integer
isDefault 是否默认：true:是 false:否 Boolean
name 用户名 String
mobile 手机 String
address 地址 String
addressName 地址名称 String
provinceId 省份ID Integer
cityId 城市ID Integer
districtId 区ID Integer
longitude 经度 BigDecimal
latitude 纬度 BigDecimal
provinceName 省份名称 String
cityName 城市名称 String
districtName 区名称 String
