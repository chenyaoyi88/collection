10.08 下单

接口调用请求说明

说明
协议 https
http请求方式 POST
请求Url https://[host]/api/v1/logistics/logisticsorder
数据格式 JSON

请求数据说明

参数 类型 是否必须 说明 验证
senderSiteName String 否 发货方地址名称
senderAddressName String 是 发货方地址详细信息
senderStreet String 否 发货方手写地址
senderX BigDecimal 是 发货方地址经度
senderY BigDecimal 是 发货方地址纬度
senderPhone String 是 发货人手机号 非空，11位手机号格式或者8位固话
senderContact String 是 发货人姓名
receiverSiteName String 否 接收方地址名称
receiverAddressName String 是 接收方地址详细信息
receiverStreet String 否 接收方手写地址
receiverX BigDecimal 是 接收方经度
receiverY BigDecimal 是 接收方纬度
receiverPhone String 是 接受方联系电话 非空，11位手机号格式或者8位固话
receiverContact String 是 接收方联系人
goodsDesc String 是 货物信息
vehicleTypeId Integer 是 车型Id（从车型列表获取）
couponCodeId Integer 否 优惠券码的ID（从优惠券列表选择的优惠券的id）
paymentType Byte 是 支付类型（1：立即支付 2:到付）
isBooking true 是 是否预约（true：预约 false:不预约）
bookingTime String 否 预约时间，当isBooking为true时，需要填这个字段(格式：yyyy-MM-dd HH:mm:ss)
insuranceStatus Byte 是 是否购买保险（0：不购买 1:购买）
insuranceOrder 对象 否 当insuranceStatus为1时，需要填该字段
listOfAdditionalRequest 对象 否 额外服务列表
type Byte 否 订单类型，默认普通订单（1：普通订单 2:循环线订单）
clothsAmount Integer 否 布匹条数，默认0条
needLoading Boolean 否 是否需要装卸，默认不需要装卸（false：不需要 true:需要）
uuid String 是 客户端生成的 UUID
halfways List 否 中途点List

扩展：insuranceOrder

参数 类型 是否必须 说明 验证
coverage BigDecimal 是 保额（目前只支持整型）
insuranceAmount BigDecimal 是 保费
applicant String 是 被保险人
content String 是 保险内容

扩展：listOfAdditionalRequest

参数 类型 是否必须 说明 验证
id Integer 是 额外服务id
name String 否 额外服务名称

扩展：

参数 类型 是否必须 说明 验证
startCityCode String 情况 发货城市code（如果选择了保险，则该字段必填）
endCityCode String 情况 收货城市code（如果选择了保险，则该字段必填）

扩展：中途点对象

参数 是否必须 说明
x 如果填中途点则为必须 纬度
y 如果填中途点则为必须 经度
phone 如果填中途点则为必须 联系人电话
siteName 非必须 地址名称
addressName 非必须 地址详细信息
street 非必须 街道
contact 非必须 联系人姓名

goodsDesc String 货物信息
vehicleType Integer 车型Id (1-小面包，2-小型箱货，3-大型箱货).
expectedMileage BigDecimal 预计里程
mileageUnit String 里程单位
actualAmount BigDecimal 实际费用
startAmount BigDecimal 起步费用
nightServiceFee BigDecimal 夜间服务费用
discount BigDecimal 折扣费用
sourceType Integer 订单来源
hsbStatus Byte 本地APP下单用的状态:
-10：已取消
-5：司机取消
5：进行中
60：已完成
status Byte 展配通系统的订单状态:
0:已取消
1:待派单(未接单)
2:已接单
3:装货中
4:配送中
5:已完成
paymentStatus Byte 支付状态:
0:未支付
10:部分支付（单独支付保险费用时的状态）
20:全额支付（全部费用支付完后的状态）
-10:已退款
paymentType Byte 支付类型:
1:立刻支付
2:货到付款
insuranceStatus Byte 保险状态:
0:不投保
1:投保
amountPaid BigDecimal 已经支付金额
paymentAmount BigDecimal 支付金额，订单实际需要支付的金额
expectedAmount BigDecimal 预计运费，调用展配通计算费用接口获取的运费
isBooking Boolean 是否预订订单：
false：非预订订单
true：预订订单
bookingTime Date 预订时间
couponCode String 优惠券码
type Byte 订单类型，默认普通订单（1：普通订单 2:循环线订单）
clothsAmount Integer 布匹条数，默认0条
needLoading Boolean 是否需要装卸，默认不需要装卸（false：不需要 true:需要）
uuid String uuid
参数 是否必须 说明
x 如果填中途点则为必须 经度
y 如果填中途点则为必须 纬度
phone 如果填中途点则为必须 联系人电话
siteName 非必须 地址名称
